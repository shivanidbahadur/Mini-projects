# Mini Calculator

A simple command-line calculator built using Python.

## Features

- Addition
- Subtraction
- Multiplication
- Division
- Division by zero handling
- Case-insensitive user input
- Perform multiple calculations without restarting the program
- Option to change input numbers

## Technologies Used

- Python 3

## Concepts Practiced

- Variables
- User Input
- Conditional Statements (`if`, `elif`, `else`)
- Loops (`while`)
- Arithmetic Operators
- Input Validation
- f-Strings
- Basic Error Handling

## How to Run

1. Open `calculator.py` in VS Code.
2. Click the **Run** button or press **Ctrl + F5**.

## Future Improvements

- Menu using numbers (1–5) instead of typing operation names.
- Calculation history.
- Scientific operations (power, square root, modulus).
- Better input validation.
- Graphical User Interface (GUI).
